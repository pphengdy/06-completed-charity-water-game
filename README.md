# Completed charity: water Game

This game project is a Jerry Can Runner, inspired by the Chrome Dino game, where the player will have to avoid running to the bacteria and collect water drops to gain points. The player will choose three difficulty modes before the start of the game: Easy, Medium, and Hard.

Technologies used: HTML, CSS, JavaScript, GitHub Copilot

While developing the game, I learned that iteration is essentially important when testing the game out to see if it works well for the end user. One of the challenges I solved in the development was the game being unplayable, and I had to debug by going through all the code to make sure that the script is up and running, and that the game is functional.
